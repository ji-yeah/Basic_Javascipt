Chapter 01. JavaScript 입문 - 02. 변수

변수와 상수는 특정 이름에 특정 값을 담을 때 사용한다.

value 라는 이름에 1 이라는 값을 넣고 console.log 를 통해 value 의 값을 출력하도록 해보면, 1이라는 값이 출력된다.

특정 이름에 특정 값을 설정하는 것을 선언 이라고 한다.

값을 선언 할 때에는 변수와 상수 두가지 종류가 있다.

변수 : 바뀔수 있는 값. 한번 값을 선언하고 나서 바꿀 수 있다.

* 변수를 선언 할 때는 let 이라는 키워드를 사용한다. (주의 : 한번 선언했으면 똑같은 이름으로 선언하지 못함.)

상수: 한번 선언하고 값이 바뀌지 않는 값. 즉, 값이 고정적이다.

상수를 선언 할 때에는 const 키워드를 사용한다.

* 상수를 선언할 때에도 한번 선언했으면 같은 이름으로 선언 할 수 없다.

* var : 변수를 선언하는 또 다른 방법으로, var 이라는 키워드가 있는데, 모던 자바스크립트에서는 더 이상 사용하지 않는다고 한다.

" IE9, IE10 같은 구형 브라우저에서는 let 과 const 를 사용 할 수 없습니다.
하지만, 보통 개발을 하게 될 때는 Babel 과 같은 도구를 사용하여 코드가 구형 브라우저에서도 돌아갈 수 있게끔 변환작업을 합니다.
만약에, 여러분이 나중에 별도의 도구 없이 구형 브라우저를 호환시켜야 하는 상황이 온다면 (그럴 일은 거의 없을 겁니다.) var 를 사용하게 될 수도 있습니다."

데이터 타입

  • - 숫자 (Number) : 그냥 바로 값을 대입하면 된다.
  • - 문자열 (String) : 텍스트 (문자열) 형태의 값은 작은 따옴표 혹은 큰 따옴표로 감싸서 선언.
    (작은 따옴표와 큰 따옴표 사용에 있어서 큰 차이는 없다.)
  • - 참/거짓 (Boolean) : boolean 은 참 혹은 거짓 두가지 종류의 값만을 나타낼 수 있다.
  • - null 과 undefined : null 은 주로 이 값이 없다! 라고 선언을 할 때 사용. undefined 는 아직 값이 설정되지 않은 것을 의미

Chapter 01. JavaScript 입문 - 03. 연산자 - 산술연산자, 대입연산자

연산자는 프로그래밍 언어에서 특정 연산을 하도록 하는 문자

산술 연산자 : 사칙연산과 같은 작업을 하는 연산자를 의미. (덧셈, 뺄셈, 곱셈, 나눗셈 등)

* ++ 는 특정 변수에 1을 바로 더해준다. ++ 가 변수 이름 앞에 오는 것과 뒤에 오는것에 차이가 있다.
console.log(a++); 를 할 때에는 1을 더하기 직전 값을 보여주고 console.log(++a); 를 할 때에는 1을 더한 다음의 값을 보여준다.
(뺄셈도 똑같이 할 수 있다.)

대입 연산자 : 특정 값에 연산을 한 값을 바로 설정 할 때 사용 할 수 있는 연산자.

Chapter 01. JavaScript 입문 - 04. 연산자 - 논리연산자

  • 논리 연산자 : 불리언 타입 (true 혹은 false)를 위한 연산자. (아래 3가지 종류)
    !: NOT
    &&: AND
    ||: OR
  • NOT 연산자 : true 는 false 로, false 는 true 로 바꿔준다.
  • AND 연산자 : 양쪽의 값이 둘 다 true 일때만 결과물이 true.
  • OR 연산자 : 양쪽의 값 중 하나라도 true 라면 결과물이 true. 두 값이 둘 다 false 일 때에만 false.

* 연산 순서 : 논리 연산자 순서는 NOT -> AND -> OR.

Chapter 01. JavaScript 입문 - 05. 연산자 - 비교연산자, 문자열 붙이기

비교 연산자 : 두 값을 비교 할 때 사용.

두 값이 일치하는지 확인 : === 는 두 값이 일치하는지 확인해준다.

두 값이 일치하지 않는지 확인 : 두 값이 일치하지 않는지 확인 할 때에는 !== 를 사용하면 된다.

크고 작음 : 두 값 중에서 무엇이 더 크고 작은지 비교하기 위해서는 다음 연산자를 사용 할 수 있다.

문자열 붙이기 : 두 문자열을 붙일 때에는 + 로 할 수 있다.

Chapter 01. JavaScript 입문 - 06. 조건문 - if, else, else if

if 문

가장 기본적인 조건문. ~하다면 ~를 해라 를 의미. 조건이 만족됐을 때 실행시킬 코드가 { } 로 감싸져 있는데, 이를 코드 블록이라고 한다.
만약에 조건이 true 가 된다면 지정한 코드가 실행되는 것이고, false 가 된다면 코드가 실행되지 않는다.

if-else 문

if-else문은 ~하다면 ~하고, 그렇지 않다면 ~해라 를 의미.

if-else if 문

if-else if 문은 여러 조건에 따라 다른 작업을 해야 할 때 사용.
